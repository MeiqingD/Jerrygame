A simple balloon popping game built with Python's turtle graphics. Click on the balloons before they reach the top of the screen to score points. 
The game tracks score, lives, and time, and features visual explosions when balloons are popped.

Features：
  Balloons spawn randomly and move upwards at varying speeds.
  Click on a balloon to pop it and score points.
  Lose a life if a balloon reaches the top of the screen.
  Countdown timer and lives display.
  Colorful explosions with multiple layers for visual effect.
  Game ends when lives run out or time is up.

Controls
  Mouse click: Pop a balloon.

Requirements
Python 3.x
turtle (standard library)
gamebase module (for line function; see tutorial reference)
